# Music Tagger

Music Tagger is a small command-line utility for viewing, editing and checking metadata in MP3 files.

## Features

* Display common ID3 tags
* Display raw ID3 frames
* Edit common tags
* Preview changes before saving
* Create backups before modifying files
* Create ID3 tags for MP3 files without metadata
* Scan directories for MP3 files
* Detect missing tags

File renaming and support for additional audio formats are not currently implemented.

## Documentation

Installation and usage instructions are available in the [application README](music-tagger/README.md).
